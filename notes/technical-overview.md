This repository provides a framework for analyzing health predictors in the Chinese elderly, using the Chinese Longitudinal Healthy Longevity Survey biomarker datasets from 2009, 2012, and 2014.

### Key Features
- Longitudinal biomarker datasets focused on health and longevity.
- Sample analyses using logistic regression in Jupyter Notebooks.
- User documentation and relevant literature included.

### Tech Stack
- Language: Jupyter Notebook (Python)
- Libraries: `pandas`, `numpy`, `matplotlib`, `seaborn`, `scikit-learn`
- Tools: Google BigQuery client, `contextily` for mapping

### Getting Started
1. Clone the repo:
   ```bash
   git clone https://github.com/justin-napolitano/chinese_longitudinal_mockup.git
   cd chinese_longitudinal_mockup
   ```
2. Install dependencies:
   ```bash
   pip install numpy pandas matplotlib seaborn scikit-learn google-cloud-bigquery contextily
   ```
3. Launch Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
4. Explore and run the notebooks.

### Gotchas
- Make sure you have Python 3.7+.
- Review the user guide in the project for effective navigation.
